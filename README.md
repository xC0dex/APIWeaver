`APIWeaver.OpenApi` is a lightweight .NET 9 library that extends the OpenAPI document generated by .NET's built-in OpenAPI document generator. It‚Äôs compatible with native AOT and offers extension methods and transformers for flexible customization.

## Features

- Authentication: Add security schemes and responses to OpenAPI documents
- Examples: Add pre-configured examples for specific types in OpenAPI documents
- Native AOT compatibility
- Other useful extension methods & transformers for OpenAPI documents, operations and schemes
